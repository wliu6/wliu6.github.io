日常行为分析——眼睛

发表于 2021-07-15 分类于 生活技巧

如何解读一个人的内心活动？人出于防卫心理，重视喜欢隐藏内心真实想法，语言的可控性可以通过锻炼提高，通过观察眼睛来分其内心OS，不失为一种巧妙又合理的方式……


视线 —— 心之所想，皆现于眼中

 胸中正，则眸子瞭焉；胸中不正，则眸子眊(mào)焉 ——《孟子》

    1. 心中正直 → 眼神笃定清明
    2. 心中不正 → 眼神游离不定
    3. 睡眼惺忪 → 懒散、不可信
    4. 眼睛雪亮、炯炯有神 → 聪明伶俐、成竹在胸
    5. 发自内心的笑 → 眼睛处在放松状态
    6. 满脸佯装微笑 → 眼睛表现为局促、不安的状态
    7. 长久对视 → 只发生于强烈的爱、恨之时

 视线的方向

    1. 向下【显示威严】
      · 眼睛向下看 + 嘴角下垂 → 显示权威、威严
      · 眼睛向下看 + 眼睛眯成一条缝 → 表达厌倦、疲倦
        · 表达厌倦常见行为 → 打哈气、看手表、反复抠手指……
    2. 水平【平等】
    3. 向上【尊敬、信赖】
      · 低头 + 眼睛向上看 + 提升眉毛 → 表现顺从、谦恭
      · 低头 + 眼睛向上看 + 压低眉毛 → 表现侵略性、攻击性、威严

 善用视线

    1. 营造良好沟通氛围：将视线落在对方眼睛与嘴巴构成的三角区域。
    2. 制造压迫感：将目光投射到对方前额三角区域。


瞳孔 —— 不受主观意志控制

  在排除「光线变化」|「散瞳药水」影响后
    1. 恐惧 → 瞳孔扩大
    2. 兴奋 → 瞳孔扩大
    3. 喜欢的事物、关注的事物 → 瞳孔扩大


眼珠 —— 最直观的观察点

 左眼

    1. 左上方 → 回想过往经历过的画面【视觉】
    2. 左中方 → 回想听到过的事情【听觉】
    3. 左下方 → 正在进行理性思考

 右眼

    1. 右上方 → 正在思考未来【视觉】
    2. 右中方 → 正在想象某种声音【听觉】
    3. 右下方 → 心思缜密+复盘ing → 狼人模式╰(°▽°)╯
   note：左撇子恰恰相反


解读目光 —— 与生俱来的能力

  无法独立分析，大多需要结合其他信息共同分析。

 东张西望

    1. 潜意识逃避、厌倦 → 寻找逃跑路线、对谈话不感兴趣。
    2. 缺乏安全感、紧张
    3. 撒谎ing

 目光斜视 —— 反应负面心态 → 不想被知晓心意

    1. 不愿看：轻蔑(不屑一顾)、得意(鄙视)、厌恶、否定、拒绝、猜疑、忧虑
    2. 不敢看：怕对方发现偷瞄一眼
    · 翻白眼 ∈ 目光斜视
      · 反例：两性交往中，女性翻白眼可能是表达爱意，需结合其他行为共同分析。

 注视

    1. 长时间注视 → 营造威胁感、压迫感、紧张气氛 → 弱势方会先将目光下垂
      · 反例：两性交往中，表达爱慕之意的眼神拉丝~
    2. 相比于无视，能降低被排斥感觉。（∵ 每个人都req一束光）
    · 善用注视~建立友好关系：6~8s的注视 + 1s左右的移开or闭眼，往复循环。

 瞪眼

    1. 恐惧
    2. 愤怒~东方人比例高
    3. 惊讶、吃惊~西方人比例高
    · 瞪眼+挑眉 → 表无辜 or 表感兴趣

 眯眼

    1. 表达不同意、暗示所有权
    2. 生气
    3. 专心致志做事
    4. 隐藏真实目的
    5. 近视
    · 识别真假笑 → 假笑嘴角上翘，真笑眼睛眯眯。

 眨眼（又称：高频率闭眼）

    1. 随意眨眼 ∈ 生理性眨眼 → 不受大脑控制
    2. 遇到潜在危害性的视觉刺激~眨眼 ∈ 保护性眨眼(生物趋利避害本能) → 不受大脑控制
    3. 自发性眨眼
      · 眨眼频次越少 → 越专心
      · 眨眼频率快且轻巧 → 大脑积极思考 → 体现兴趣浓厚
      · 眨眼频次变少 + 闭眼时间变长 → 不耐烦、有厌烦情绪、感觉无趣味……

 闭眼

    1. 拒绝观看 ∈ 保护性生物行为
    2. 隔绝视觉干扰 → 提升大脑思考效率【抬头看天花板、抬头看填空类似效果】
    3. 闭眼说话的人 → 大多是言而无信之人


目光识人

  1. 视线不集中在对方，被对方注视立刻移开视线 → 大都为内向者，大都有自卑感

  2. 斜眼看 → 非常有兴趣，但不想让对方识破

  3. 看异性一眼，随后故意移开视线 → 有很强的兴趣

  4. 目光闪烁、飘逸不定 + 年轻人 → 心思活络，好主意、鬼点子多

  5. 目光闪烁、飘逸不定 + 上了年纪 → 老谋深算、老奸巨猾的阴谋家

  6. 长时间向下看 → 易导致抑郁 → 低头向下看【恶性循环】
    · 缓解方式：改善低头看习惯，多平视、远望。

  7. 眨眼频率先下降，再大幅上升，最后变慢 → 典型撒谎标志（∵ 不受大脑控制）
